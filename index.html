Built & shipped

Open-source plugins for Paper Minecraft servers, plus tooling for plugin developers. Every repo below is public on GitHub.

ChatPing Discord-style @mention pings for Paper Minecraft servers. Java
OptiCam Realistic CCTV camera system for Paper Minecraft servers. Java
ScappiSupport In-game support tickets bridged to Discord threads. Java
TOSGuard TOS/rules acceptance gate with GUI and version control. Java
ScappiCore Shared foundation library — async storage, GUIs, Discord webhooks. Java
ScappiWatch Discord audit & alerts — joins, deaths, TPS, anomalies, bot mode. Java
ScappiPunish Bans, mutes, warns with escalation ladders and cross-server sync. Java
ScappiStaff Staff mode with crash-safe inventories, vanish, freeze and examine. Java
ScappiChat Modular chat suite — filter, anti-spam, channels, staff chat, mentions. Java
ScappiPass Battle pass & quests — 30+ triggers, zero-DB event path, anti-cheese. Java
MinecraftObfuscator Obfuscator for your own plugin JARs (Python GUI + Java/ASM engine). Python
